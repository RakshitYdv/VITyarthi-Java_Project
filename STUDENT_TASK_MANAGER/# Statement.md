# Statement

## Problem Statement
Students need a simple console tool to manage academic tasks and deadlines locally.

## Scope
Single-user local console application with persistent storage in CSV.

## Target users
Students, teachers, small groups.

## High-level features
- Add/Edit/Delete tasks
- Mark complete/uncomplete
- List/filter tasks, upcoming tasks
- Export/Import CSV
- Backup Data
